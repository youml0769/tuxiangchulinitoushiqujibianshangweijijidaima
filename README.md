# 图像处理逆透视、去畸变上位机及代码

## 资源描述

本仓库提供了一个能够同时完成去畸变与逆透视或去畸变+逆透视图像处理的上位机软件及相关代码。该软件名为RUBO-UDIPM，遵循GPLv3协议，是开源的。软件仅用于竞赛、学习和交流，不用于商业或付费教学目的。

1. **下载与安装**：请从本仓库下载资源文件，并按照提供的安装指南进行安装。
